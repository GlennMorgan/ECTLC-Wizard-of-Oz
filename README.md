# ECTLC Wizard of Oz Production Hub

This folder is ready to upload to a GitHub repository and publish with GitHub Pages.

## Files

- `index.html` - public landing page
- `revised-script.html` - interactive revised ECTLC script
- `assets/storyboard-page-1.png` - storyboard page 1
- `assets/storyboard-page-2.png` - storyboard page 2
- `planning-session.html` - included only if available

## Simple GitHub Pages setup

1. Create a new GitHub repository.
2. Upload all files and folders from this package.
3. Commit the files.
4. Go to repository Settings.
5. Go to Pages.
6. Under Build and deployment, choose Deploy from a branch.
7. Choose `main` and `/root`.
8. Save.
9. Wait a few minutes for GitHub to publish the site.

Your site URL will usually be:

`https://<your-github-username>.github.io/<repository-name>/`
